场景二

好家伙！文件中使用了大量不规范的下划线命名，如何一步将其全变为驼峰？

借助 IDE 的正则替换功能（如果你的编辑器没有这个功能， 那你就复制出来用 string 操作吧）

Webstorm

  1. 按 shift + command + R，进入替换模式
  2. 选择正则替换
  3. 输入 _(\w) 替换为 \U$1
  4. 替换即可

Tips

替换为小写则是：

\L$1
