this 是一个动态指针，它指向当前作用域对象，如果当前定义对象的作用域没有发生变化，那么this就会指向当前对象。
This可以存在于任何位置，并不局限与对象、方法内，也可以是其他特殊的上下文环境。

  3. 函数的引用和调用中this的变数
  4. call 和 apply 对 this的影响
  5. 原型继承中的 This
  6. 异步调用的 This
  7. 定时器中的 This
